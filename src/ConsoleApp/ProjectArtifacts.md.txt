Перечень артефактов проекта Lab-01-Огнев

Исходный код
- Program.cs — основной файл вызова модулей
-Содержит метод Main() - стартовую точку выполнения программы-
-Выполняет последовательную координацию работы всех модулей:
1. Получает входные данные через InputModule.GetInput()
2. Обрабатывает данные через ProcessingModule.CountRussianVowels()
3. Выводит результат на консоль-
-Работает как диспетчер который связывает все компоненты системы-
- Modules/InputModule.cs — модуль ввода данных
-Содержит статический метод GetInput() который:
1. Выводит приглашение на русском языке: "Введите строку: "
2. Читает введёную строку из консоли Console.ReadLine()
3. Возвращает полученную строку вызывающему коду-
- Modules/ValidationModule.cs — модуль проверки данных
-Содержит статический метод IsValidString(string text) который:
1. Проверяет что строка не является NULL
2. Проверяет что строка не является пустой (string.empty)
3. Использует встроенный метод string.IsNullOrEmpty() для комплексной проверки
4. Возвращает true если строка валидна, false если нет-
- Modules/ProcessingModule.cs — модуль обработки и вывода данных
-Содержит метод CountRussianVowels(string text), который выполняет:
1. Предворительную проверку (строка 6):
if (!ValidationModule.IsValidString(text))
            return 0;
-Использует ValidationModule для проверки данных-
-Если строка невалидна сразу возвращает 0-
2. Определение глассных букв (строка 9):
  char[] vowels = { 'а', 'е', 'ё', 'и', 'о', 'у', 'ы', 'э', 'ю', 'я' };
-Создает массив всех русских глассных букв-
-Включает букву 'ё', которая часто забывается-
-Использует строчные буквы для сравнения-
3. Основной алгоритм подсчета (строки 13-21):
 foreach (char c in text)
            {
                char LowerChar = char.ToLower(c);

                if (Array.IndexOf(vowels, LowerChar) >= 0)
                {
                    count++;
                }
            }
-Проходит по каждому символу строки-
-Приводит символ к нижнему регистру для регистронезависимого сравнения-
-Проверяет наличие символов в массиве гласных через Array.IndexOf()-
-Увеличевает счетчик при совпадении-
4. Возврат результата (строка 22):
return count;
-Возвращает общее кол-во найденных глассных-
